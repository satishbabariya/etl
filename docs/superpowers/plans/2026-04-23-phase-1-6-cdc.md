# Phase I.6 — Postgres CDC Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Postgres logical-replication CDC as a third sync mode alongside the Phase I.2 cursor-incremental and Phase I.3 WASM modes, with a parent/child Temporal workflow topology, capture-and-stream snapshot handoff, in-band schema change events, and an append-only Parquet apply-change-stream loader.

**Architecture:** A new `SyncMode::Cdc` flavor on `PostgresSourceSpec` selects CDC at dispatch time. Slot lifecycle (ensure/advance/release) is a new `cdc_slots` catalog table plus SQL against `pg_create_logical_replication_slot` / `pg_replication_slot_advance` / `pg_drop_replication_slot`. The CDC pipeline runs as `CdcPipelineWorkflow` (long-lived parent, uses `continue-as-new` every 10M events or at the activity-count budget) that first launches a `CdcSnapshotWorkflow` child (finite, chunked reads pinned to the slot's `CONSISTENT_POINT` LSN, emits rows with `_cdc.op = "s"`) then enters a streaming loop calling a `cdc_read_window` activity — each window opens a pgoutput replication stream, decodes Insert/Update/Delete/Relation/Begin/Commit messages into a `RecordBatch` carrying per-row `_cdc.op`, `_cdc.lsn`, `_cdc.commit_ts`, `_cdc.txid` metadata, writes through Phase I.5's transform DAG, and lands in the new `cdc_parquet` loader which appends one event per row (no in-place updates — compaction deferred to Phase II).

**Tech Stack:** `tokio-postgres` 0.7 with the `replication` feature for COPY BOTH / pgoutput; unchanged elsewhere — Rust 1.88, arrow 53, temporalio-sdk 0.2, wasmtime 26, sqlx 0.8, blake3.

```rust
    Int64,
    TimestampTz,
    Lsn,

pub fn lsn_to_string(lsn: u64) -> String {
    format!("{:X}/{:X}", lsn >> 32, lsn as u32)
}

pub fn lsn_from_string(s: &str) -> anyhow::Result<u64> {
    let (hi, lo) = s
        .split_once('/')
        .ok_or_else(|| anyhow::anyhow!("LSN missing '/': {s}"))?;
    let hi: u64 = u64::from_str_radix(hi, 16)?;
    let lo: u64 = u64::from_str_radix(lo, 16)?;
    Ok((hi << 32) | lo)
}
```

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum SlotState {
    Active,
    Paused,
    Released,
}

#[derive(Debug, Clone)]
pub struct CdcSlot {
    pub pipeline_id: PipelineId,
    pub slot_name: String,
    pub publication_name: String,
    pub consistent_point: String,
    pub confirmed_flush: Option<String>,
    pub state: SlotState,
}

pub async fn upsert(pool: &PgPool, slot: &CdcSlot) -> sqlx::Result<()> {
    let state_s = match slot.state {
        SlotState::Active => "active",
        SlotState::Paused => "paused",
        SlotState::Released => "released",
    };
    sqlx::query(
        "INSERT INTO cdc_slots(pipeline_id, slot_name, publication_name, \
          consistent_point, confirmed_flush, state, updated_at) \
         VALUES ($1,$2,$3,$4,$5,$6, now()) \
         ON CONFLICT (pipeline_id) DO UPDATE SET \
           slot_name=EXCLUDED.slot_name, \
           publication_name=EXCLUDED.publication_name, \
           consistent_point=EXCLUDED.consistent_point, \
           confirmed_flush=COALESCE(EXCLUDED.confirmed_flush, cdc_slots.confirmed_flush), \
           state=EXCLUDED.state, \
           updated_at=now()",
    )
    .bind(slot.pipeline_id.as_uuid())
    .bind(&slot.slot_name)
    .bind(&slot.publication_name)
    .bind(&slot.consistent_point)
    .bind(&slot.confirmed_flush)
    .bind(state_s)
    .execute(pool)
    .await?;
    Ok(())
}

pub async fn get(pool: &PgPool, pipeline_id: PipelineId) -> sqlx::Result<Option<CdcSlot>> {
    let row: Option<(uuid::Uuid, String, String, String, Option<String>, String)> = sqlx::query_as(
        "SELECT pipeline_id, slot_name, publication_name, consistent_point, confirmed_flush, state \
         FROM cdc_slots WHERE pipeline_id = $1",
    )
    .bind(pipeline_id.as_uuid())
    .fetch_optional(pool)
    .await?;
    Ok(row.map(|(pid, sn, pn, cp, cf, st)| CdcSlot {
        pipeline_id: PipelineId::from_uuid_unchecked(pid),
        slot_name: sn,
        publication_name: pn,
        consistent_point: cp,
        confirmed_flush: cf,
        state: match st.as_str() {
            "active" => SlotState::Active,
            "paused" => SlotState::Paused,
            "released" => SlotState::Released,
            other => panic!("unknown slot state {other}"),
        },
    }))
}

pub async fn update_confirmed_flush(
    pool: &PgPool,
    pipeline_id: PipelineId,
    lsn: &str,
) -> sqlx::Result<()> {
    sqlx::query(
        "UPDATE cdc_slots SET confirmed_flush=$1, updated_at=now() WHERE pipeline_id=$2",
    )
    .bind(lsn)
    .bind(pipeline_id.as_uuid())
    .execute(pool)
    .await?;
    Ok(())
}
```

`crates/worker/src/connectors/postgres/cdc/decode.rs`:

```rust
use anyhow::{anyhow, Context};
use byteorder::{BigEndian, ReadBytesExt};
use std::collections::HashMap;
use std::io::Read;

#[derive(Debug, Clone, PartialEq)]
pub struct RelationInfo {
    pub rel_id: u32,
    pub namespace: String,
    pub name: String,
    pub columns: Vec<ColumnInfo>,
}

#[derive(Debug, Clone, PartialEq)]
pub struct ColumnInfo {
    pub name: String,
    pub type_oid: u32,
    pub is_key: bool,
}

#[derive(Debug, Clone, PartialEq)]
pub enum CdcEvent {
    Begin { final_lsn: u64, commit_ts_micros: i64, xid: u32 },
    Commit { commit_lsn: u64, end_lsn: u64, commit_ts_micros: i64 },
    Relation(RelationInfo),
    Insert { rel_id: u32, row: Vec<Option<String>> },
    Update { rel_id: u32, row: Vec<Option<String>> },
    Delete { rel_id: u32, key: Vec<Option<String>> },
}

/// Decodes a single pgoutput XLogData message body (the bytes after the
/// `w` header has been stripped by the caller).
pub fn decode_message(body: &[u8]) -> anyhow::Result<CdcEvent> {
    let mut rdr = &body[..];
    let tag = rdr.read_u8().context("reading message tag")?;
    match tag {
        b'B' => decode_begin(rdr),
        b'C' => decode_commit(rdr),
        b'R' => decode_relation(rdr),
        b'I' => decode_insert(rdr),
        b'U' => decode_update(rdr),
        b'D' => decode_delete(rdr),
        other => Err(anyhow!("unsupported pgoutput tag {other}")),
    }
}

fn decode_begin(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let final_lsn = rdr.read_u64::<BigEndian>()?;
    let commit_ts = rdr.read_i64::<BigEndian>()?;
    let xid = rdr.read_u32::<BigEndian>()?;
    Ok(CdcEvent::Begin { final_lsn, commit_ts_micros: commit_ts, xid })
}

fn decode_commit(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let _flags = rdr.read_u8()?;
    let commit_lsn = rdr.read_u64::<BigEndian>()?;
    let end_lsn = rdr.read_u64::<BigEndian>()?;
    let commit_ts = rdr.read_i64::<BigEndian>()?;
    Ok(CdcEvent::Commit { commit_lsn, end_lsn, commit_ts_micros: commit_ts })
}

fn read_cstr(rdr: &mut &[u8]) -> anyhow::Result<String> {
    let nul = rdr.iter().position(|&b| b == 0)
        .ok_or_else(|| anyhow!("cstring missing NUL"))?;
    let s = std::str::from_utf8(&rdr[..nul])?.to_owned();
    *rdr = &rdr[nul + 1..];
    Ok(s)
}

fn decode_relation(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let rel_id = rdr.read_u32::<BigEndian>()?;
    let namespace = read_cstr(&mut rdr)?;
    let name = read_cstr(&mut rdr)?;
    let _replica_identity = rdr.read_u8()?;
    let n_cols = rdr.read_u16::<BigEndian>()? as usize;
    let mut columns = Vec::with_capacity(n_cols);
    for _ in 0..n_cols {
        let flags = rdr.read_u8()?;
        let name = read_cstr(&mut rdr)?;
        let type_oid = rdr.read_u32::<BigEndian>()?;
        let _type_mod = rdr.read_i32::<BigEndian>()?;
        columns.push(ColumnInfo {
            name,
            type_oid,
            is_key: (flags & 0x01) != 0,
        });
    }
    Ok(CdcEvent::Relation(RelationInfo { rel_id, namespace, name, columns }))
}

fn decode_tuple(rdr: &mut &[u8]) -> anyhow::Result<Vec<Option<String>>> {
    let n = rdr.read_u16::<BigEndian>()? as usize;
    let mut out = Vec::with_capacity(n);
    for _ in 0..n {
        let kind = rdr.read_u8()?;
        match kind {
            b'n' => out.push(None),
            b'u' => out.push(None), // TOAST unchanged — treat as null for Phase I.6
            b't' => {
                let len = rdr.read_u32::<BigEndian>()? as usize;
                let mut buf = vec![0u8; len];
                rdr.read_exact(&mut buf)?;
                out.push(Some(String::from_utf8(buf)?));
            }
            other => return Err(anyhow!("unknown tuple col kind {other}")),
        }
    }
    Ok(out)
}

fn decode_insert(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let rel_id = rdr.read_u32::<BigEndian>()?;
    let tag = rdr.read_u8()?;
    anyhow::ensure!(tag == b'N', "insert tuple tag expected 'N', got {tag}");
    let row = decode_tuple(&mut rdr)?;
    Ok(CdcEvent::Insert { rel_id, row })
}

fn decode_update(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let rel_id = rdr.read_u32::<BigEndian>()?;
    // May have K (key-only before image) or O (old tuple). Skip both.
    let tag = rdr.read_u8()?;
    let rdr2: &mut &[u8] = &mut rdr;
    if tag == b'K' || tag == b'O' {
        let _skipped = decode_tuple(rdr2)?;
        let _n_tag = rdr2.read_u8()?;
    }
    let row = decode_tuple(rdr2)?;
    Ok(CdcEvent::Update { rel_id, row })
}

fn decode_delete(mut rdr: &[u8]) -> anyhow::Result<CdcEvent> {
    let rel_id = rdr.read_u32::<BigEndian>()?;
    let _tag = rdr.read_u8()?; // K or O
    let key = decode_tuple(&mut rdr)?;
    Ok(CdcEvent::Delete { rel_id, key })
}
```

```rust
pub async fn ensure_publication(conn_url: &str, publication: &str, table: &str) -> anyhow::Result<()> {
    let mut c = PgConnection::connect(conn_url).await?;
    let exists: (bool,) = sqlx::query_as(

    )
    .bind(publication)
    .fetch_one(&mut c)
    .await?;
    if !exists.0 {
        let stmt = format!("CREATE PUBLICATION \"{publication}\" FOR TABLE \"{table}\"");
        sqlx::query(&stmt).execute(&mut c).await.context("CREATE PUBLICATION")?;
    }
    Ok(())
}

pub async fn ensure_slot(conn_url: &str, slot: &str) -> anyhow::Result<EnsureResult> {
    let mut c = PgConnection::connect(conn_url).await?;
    let existing: Option<(String,)> = sqlx::query_as(
        "SELECT confirmed_flush_lsn::text FROM pg_replication_slots WHERE slot_name = $1",
    )
    .bind(slot)
    .fetch_optional(&mut c)
    .await?;
    if let Some((lsn,)) = existing {
        return Ok(EnsureResult {
            slot_name: slot.to_string(),
            publication_name: String::new(), // filled in by caller
            consistent_point: lsn,
            created: false,
        });
    }
    let row: (String, String) = sqlx::query_as(
        "SELECT slot_name, lsn::text \
         FROM pg_create_logical_replication_slot($1, 'pgoutput')",
    )
    .bind(slot)
    .fetch_one(&mut c)
    .await
    .context("pg_create_logical_replication_slot")?;
    Ok(EnsureResult {
        slot_name: row.0,
        publication_name: String::new(),
        consistent_point: row.1,
        created: true,
    })
}

    sqlx::query("SELECT pg_drop_replication_slot($1)")
        .bind(slot)
        .execute(&mut c)
        .await
        .context("pg_drop_replication_slot")?;
    Ok(())
}

    .bind(slot)
    .fetch_one(&mut c)

    Ok(lag)
}
```

## Task 8: Snapshot with pinned LSN

**Files:**

- [ ] **Step 1: Write snapshot helper**

The snapshot uses the slot's `CONSISTENT_POINT` + exported snapshot name. For MVP we use the simpler pattern: create slot → read `confirmed_flush_lsn` as consistent point → snapshot with `REPEATABLE READ` isolation immediately after. Postgres guarantees the slot holds the LSN; any row present at that point is visible.

```rust
use anyhow::Context;
use arrow::array::{ArrayRef, Int64Builder, StringBuilder, TimestampMicrosecondBuilder};
use arrow::datatypes::{DataType, Field, Schema, SchemaRef, TimeUnit};
use arrow::record_batch::RecordBatch;
use sqlx::{Connection, PgConnection, Row};
use std::sync::Arc;

pub struct SnapshotChunk {
    pub batch: RecordBatch,
    pub is_final: bool,
    pub last_pk: Option<i64>,
}

/// Reads one chunk of rows where pk > last_pk, ordered by pk, limit batch_size.
/// Caller supplies a fresh REPEATABLE READ transaction.
pub async fn read_chunk(
    conn_url: &str,
    schema: &str,
    table: &str,
    pk_col: &str,
    last_pk: Option<i64>,
    batch_size: usize,
    consistent_point: &str,
    cdc_schema: SchemaRef,
) -> anyhow::Result<SnapshotChunk> {
    let mut c = PgConnection::connect(conn_url).await?;
    sqlx::query("BEGIN ISOLATION LEVEL REPEATABLE READ READ ONLY")
        .execute(&mut c).await?;
    let where_clause = match last_pk {
        Some(_) => format!(" WHERE \"{pk_col}\" > $1"),
        None => String::new(),
    };
    let stmt = format!(
        "SELECT * FROM \"{schema}\".\"{table}\"{where_clause} ORDER BY \"{pk_col}\" LIMIT {batch_size}"
    );
    let mut q = sqlx::query(&stmt);
    if let Some(pk) = last_pk {
        q = q.bind(pk);
    }
    let rows = q.fetch_all(&mut c).await?;
    sqlx::query("COMMIT").execute(&mut c).await?;

    let (batch, last) =
        rows_to_cdc_batch(rows, pk_col, cdc_schema, consistent_point)
            .context("rows → cdc batch")?;
    let is_final = batch.num_rows() < batch_size;
    Ok(SnapshotChunk { batch, is_final, last_pk: last })
}

/// Build a snapshot RecordBatch with the user columns as text plus the
/// four `_cdc.*` metadata columns. All rows share `_cdc.op = "s"` and
/// `_cdc.lsn = consistent_point`.
fn rows_to_cdc_batch(
    rows: Vec<sqlx::postgres::PgRow>,
    pk_col: &str,
    schema: SchemaRef,
    consistent_point: &str,
) -> anyhow::Result<(RecordBatch, Option<i64>)> {
    use arrow::array::StringArray;
    let n = rows.len();
    let mut last_pk: Option<i64> = None;
    // All data columns are rendered as String for MVP — type fidelity is
    // handled by the non-CDC path via sqlx_to_arrow; CDC MVP lands TEXT.
    let data_fields: Vec<&Field> = schema
        .fields()
        .iter()
        .filter(|f| !f.name().starts_with("_cdc"))
        .map(|f| f.as_ref())
        .collect();
    let mut col_builders: Vec<StringBuilder> =
        data_fields.iter().map(|_| StringBuilder::new()).collect();
    let mut op_b = StringBuilder::new();
    let mut lsn_b = StringBuilder::new();
    let mut ts_b = TimestampMicrosecondBuilder::new();
    let mut tx_b = Int64Builder::new();
    for r in &rows {
        for (i, f) in data_fields.iter().enumerate() {
            let v: Option<String> = r.try_get::<Option<String>, _>(f.name().as_str()).ok().flatten();
            col_builders[i].append_option(v);
        }
        op_b.append_value("s");
        lsn_b.append_value(consistent_point);
        ts_b.append_null();
        tx_b.append_null();
        if let Ok(pk) = r.try_get::<i64, _>(pk_col) {
            last_pk = Some(pk);
        }
    }
    let mut cols: Vec<ArrayRef> = col_builders
        .into_iter()
        .map(|mut b| Arc::new(b.finish()) as ArrayRef)
        .collect();
    cols.push(Arc::new(op_b.finish()));
    cols.push(Arc::new(lsn_b.finish()));
    cols.push(Arc::new(ts_b.finish()));
    cols.push(Arc::new(tx_b.finish()));
    let batch = RecordBatch::try_new(schema, cols)?;
    Ok((batch, last_pk))
}

pub fn cdc_schema_for(data_cols: &[(&str, DataType)]) -> SchemaRef {
    let mut fields: Vec<Field> = data_cols
        .iter()
        .map(|(n, t)| Field::new(*n, t.clone(), true))
        .collect();
    fields.push(Field::new(common_types::cdc::COL_OP, DataType::Utf8, false));
    fields.push(Field::new(common_types::cdc::COL_LSN, DataType::Utf8, false));
    fields.push(Field::new(
        common_types::cdc::COL_COMMIT_TS,
        DataType::Timestamp(TimeUnit::Microsecond, Some("UTC".into())),
        true,
    ));
    fields.push(Field::new(common_types::cdc::COL_TXID, DataType::Int64, true));
    Arc::new(Schema::new(fields))
}
```

```rust
use anyhow::{Context, Result};
use std::time::Duration;
use tokio_postgres::{Client, Config, NoTls};

use super::decode::{self, CdcEvent, RelationInfo, RelationTable};
use common_types::cursor::{lsn_from_string, lsn_to_string};

pub struct WindowOutput {
    pub events: Vec<CdcEvent>,
    pub relations: RelationTable,
    pub new_position: Option<u64>,
    pub is_empty: bool,
}

pub async fn read_window(
    conn_url: &str,
    slot_name: &str,
    publication: &str,
    start_lsn: Option<&str>,
    max_events: usize,
    max_wait: Duration,
    mut relations: RelationTable,
) -> Result<WindowOutput> {
    let cfg: Config = conn_url.parse()?;
    // tokio-postgres requires the replication param directly in the URL/options:
    let mut repl_cfg = cfg.clone();
    repl_cfg.replication_mode(tokio_postgres::config::ReplicationMode::Logical);
    let (client, conn) = repl_cfg.connect(NoTls).await.context("replication connect")?;
    tokio::spawn(async move {
        let _ = conn.await;
    });
    let start = start_lsn.unwrap_or("0/0");
    let opts = format!("(\"proto_version\" '1', \"publication_names\" '{publication}')");
    let query = format!("START_REPLICATION SLOT \"{slot_name}\" LOGICAL {start} {opts}");
    let copy_stream = client.copy_both_simple::<bytes::Bytes>(&query).await
        .context("START_REPLICATION")?;
    let deadline = tokio::time::Instant::now() + max_wait;
    let mut out: Vec<CdcEvent> = Vec::new();
    let mut last_end_lsn: Option<u64> = None;
    use futures_util::StreamExt;
    let mut stream = Box::pin(copy_stream);
    while out.len() < max_events {
        let remaining = deadline.saturating_duration_since(tokio::time::Instant::now());
        if remaining.is_zero() { break; }
        let msg = match tokio::time::timeout(remaining, stream.next()).await {
            Ok(Some(Ok(b))) => b,
            Ok(Some(Err(e))) => return Err(e.into()),
            Ok(None) => break,
            Err(_) => break,
        };
        // The first byte is the replication message type.
        if msg.is_empty() { continue; }
        match msg[0] {
            b'w' => {
                // XLogData: w | start_lsn (u64 BE) | end_lsn | ts | payload
                if msg.len() < 25 { continue; }
                let end_lsn = u64::from_be_bytes(msg[9..17].try_into().unwrap());
                let body = &msg[25..];
                let ev = decode::decode_message(body)?;
                if let CdcEvent::Relation(r) = &ev {
                    relations.insert(r.rel_id, r.clone());
                }
                out.push(ev);
                last_end_lsn = Some(end_lsn);
            }
            b'k' => {
                // Primary keepalive — ignore; we don't reply yet in MVP.
            }
            _ => {}
        }
    }
    let is_empty = out.is_empty();
    Ok(WindowOutput { events: out, relations, new_position: last_end_lsn, is_empty })
}

/// Convert a flush of `read_window` into a `RecordBatch` ready for the
/// transform/load pipeline.
pub fn events_to_batch(
    events: &[CdcEvent],
    relations: &RelationTable,
    rel_id_filter: u32,
    schema: arrow::datatypes::SchemaRef,
) -> anyhow::Result<arrow::record_batch::RecordBatch> {
    use arrow::array::{ArrayRef, Int64Builder, StringBuilder, TimestampMicrosecondBuilder};
    use std::sync::Arc;
    let rel = relations
        .get(&rel_id_filter)
        .ok_or_else(|| anyhow::anyhow!("no Relation seen for rel_id {rel_id_filter}"))?;
    let n_data_cols = rel.columns.len();
    let mut col_builders: Vec<StringBuilder> =
        (0..n_data_cols).map(|_| StringBuilder::new()).collect();
    let mut op_b = StringBuilder::new();
    let mut lsn_b = StringBuilder::new();
    let mut ts_b = TimestampMicrosecondBuilder::new();
    let mut tx_b = Int64Builder::new();
    let mut current_txid: Option<u32> = None;
    let mut current_commit_ts: Option<i64> = None;
    let mut current_lsn: Option<u64> = None;
    for ev in events {
        match ev {
            CdcEvent::Begin { xid, commit_ts_micros, .. } => {
                current_txid = Some(*xid);
                current_commit_ts = Some(*commit_ts_micros);
            }
            CdcEvent::Commit { end_lsn, .. } => {
                current_lsn = Some(*end_lsn);
            }
            CdcEvent::Relation(_) => {}
            CdcEvent::Insert { rel_id, row } if *rel_id == rel_id_filter => {
                for (i, v) in row.iter().enumerate() {
                    col_builders[i].append_option(v.as_deref());
                }
                op_b.append_value("i");
                lsn_b.append_value(lsn_to_string(current_lsn.unwrap_or(0)));
                match current_commit_ts {
                    Some(ts) => ts_b.append_value(ts),
                    None => ts_b.append_null(),
                }
                match current_txid {
                    Some(t) => tx_b.append_value(t as i64),
                    None => tx_b.append_null(),
                }
            }
            CdcEvent::Update { rel_id, row } if *rel_id == rel_id_filter => {
                for (i, v) in row.iter().enumerate() {
                    col_builders[i].append_option(v.as_deref());
                }
                op_b.append_value("u");
                lsn_b.append_value(lsn_to_string(current_lsn.unwrap_or(0)));
                tx_b.append_option(current_txid.map(|t| t as i64));
                ts_b.append_option(current_commit_ts);
            }
            CdcEvent::Delete { rel_id, key } if *rel_id == rel_id_filter => {
                // For delete, only key columns are populated; others NULL.
                for (i, col) in rel.columns.iter().enumerate() {
                    if col.is_key {
                        col_builders[i].append_option(key.get(i).and_then(|v| v.as_deref()));
                    } else {
                        col_builders[i].append_null();
                    }
                }
                op_b.append_value("d");
                lsn_b.append_value(lsn_to_string(current_lsn.unwrap_or(0)));
                tx_b.append_option(current_txid.map(|t| t as i64));
                ts_b.append_option(current_commit_ts);
            }
            _ => {}
        }
    }
    let mut cols: Vec<ArrayRef> = col_builders
        .into_iter()
        .map(|mut b| Arc::new(b.finish()) as ArrayRef)
        .collect();
    cols.push(Arc::new(op_b.finish()));
    cols.push(Arc::new(lsn_b.finish()));
    cols.push(Arc::new(ts_b.finish()));
    cols.push(Arc::new(tx_b.finish()));
    Ok(arrow::record_batch::RecordBatch::try_new(schema, cols)?)
}
```

```rust
            publication_name: pub_name,

        self: Arc<Self>,
        _ctx: ActivityContext,
        input: SnapshotChunkInput,

        let cdc_schema = snapshot::cdc_schema_for(&[(input.pk_col.as_str(), DataType::Utf8)]);
        let chunk = snapshot::read_chunk(
            &input.source_url, &input.schema, &input.table,
            &input.pk_col, input.last_pk, input.batch_size,
            &input.consistent_point, cdc_schema,

                .write(&input.destination, input.pipeline_id, input.run_id,
                       input.batch_seq, &chunk.batch)
                .await
                .map_err(retryable)?;
        }
        Ok(SnapshotChunkOutput {
            rows: chunk.batch.num_rows(),
            is_final: chunk.is_final,
            last_pk: chunk.last_pk,
        })
    }

        self: Arc<Self>,
        _ctx: ActivityContext,
        input: ReadWindowInput,

        use std::time::Duration;
        let out = stream::read_window(
            &input.source_url, &input.slot_name, &input.publication_name,
            input.start_lsn.as_deref(), input.max_events,
            Duration::from_millis(input.max_wait_ms),
            Default::default(),
        ).await.map_err(retryable)?;
        if out.is_empty {
            return Ok(ReadWindowOutput { rows: 0, new_lsn: None });
        }
        let rel_id = out.relations.values()
            .find(|r| format!("{}.{}", r.namespace, r.name) == input.table_rel_name)
            .map(|r| r.rel_id)
            .ok_or_else(|| retryable(anyhow::anyhow!("no Relation for {}", input.table_rel_name)))?;
        let rel = out.relations.get(&rel_id).unwrap();
        let cols: Vec<(&str, DataType)> =
            rel.columns.iter().map(|c| (c.name.as_str(), DataType::Utf8)).collect();
        let schema = snapshot::cdc_schema_for(&cols);
        let batch = stream::events_to_batch(&out.events, &out.relations, rel_id, schema)
            .map_err(retryable)?;
        let rows = batch.num_rows();
        if rows > 0 {
            CdcParquetLoader
                .write(&input.destination, input.pipeline_id, input.run_id,
                       input.batch_seq, &batch)
                .await
                .map_err(retryable)?;
        }
        let new_lsn = out.new_position.map(common_types::cursor::lsn_to_string);
        if let Some(lsn) = &new_lsn {
            let pid = PipelineId::from_uuid_unchecked(input.pipeline_id);
            catalog::cdc::update_confirmed_flush(self.catalog.pool(), pid, lsn)
                .await.map_err(|e| retryable(anyhow::anyhow!(e)))?;
        }
        Ok(ReadWindowOutput { rows, new_lsn })
    }

    #[activity]
    pub async fn release_slot(
        self: Arc<Self>,
        _ctx: ActivityContext,
        input: ReleaseSlotInput,
    ) -> Result<(), ActivityError> {
        let _ = slot::release_slot(&input.source_url, &input.slot_name).await;
        Ok(())
```

```rust
use arrow::record_batch::RecordBatch;
use common_types::pipeline_spec::DestinationSpec;
use parquet::arrow::ArrowWriter;
use parquet::file::properties::WriterProperties;
use std::path::PathBuf;
use uuid::Uuid;

pub struct CdcParquetLoader;

impl CdcParquetLoader {
    pub async fn write(
        &self,
        dest: &DestinationSpec,
        pipeline_id: Uuid,
        run_id: Uuid,
        batch_seq: u32,
        batch: &RecordBatch,
    ) -> Result<PathBuf> {
        let base = match dest {
            DestinationSpec::LocalParquet(s) => s.base_path.clone(),
        };
        let mut path = PathBuf::from(&base);
        path.push(pipeline_id.to_string());
        path.push("cdc");
        path.push(run_id.to_string());
        std::fs::create_dir_all(&path)
            .with_context(|| format!("create dir {}", path.display()))?;
        path.push(format!("batch-{:05}.parquet", batch_seq));
        let file = std::fs::File::create(&path)
            .with_context(|| format!("create {}", path.display()))?;
        let props = WriterProperties::builder().build();
        let mut w = ArrowWriter::try_new(file, batch.schema(), Some(props))?;
        w.write(batch)?;
        w.close()?;
        Ok(path)
    }
}
```

```rust
use common_types::pipeline_spec::{DestinationSpec, PipelineSpec, SourceSpec};
use serde::{Deserialize, Serialize};
use std::time::Duration;
use temporalio_macros::{workflow, workflow_methods};
use temporalio_sdk::{
    ActivityOptions, ChildWorkflowOptions, WorkflowContext, WorkflowContextView,
    WorkflowResult, WorkflowTermination, ContinueAsNewWorkflowExecution,
};
use uuid::Uuid;

use crate::activities::cdc::CdcActivities;
use crate::activities::cdc::inputs::{
    EnsureSlotInput, EnsureSlotOutput, ReadWindowInput, ReadWindowOutput,
};
use crate::activities::run_lifecycle::{FailRunInput, RunLifecycleActivities};
use crate::workflows::cdc_snapshot::{CdcSnapshotInput, CdcSnapshotWorkflow};

/// Trigger `continue-as-new` once we've processed this many events
/// (mirrors RFC-8 §`continue-as-new policy for CDC`, conservative MVP).
const CAN_EVENT_BUDGET: u64 = 10_000_000;

#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct CdcPipelineInput {
    pub run_id: Uuid,
    pub pipeline_id: Uuid,
    pub tenant_id: Uuid,
    pub spec: PipelineSpec,
    pub source_url: String,
    /// None on first launch; Some(lsn) on continue-as-new.
    pub resume_lsn: Option<String>,
    /// None until snapshot has run once (first launch); Some after.
    pub slot_ready: Option<EnsureSlotOutput>,
    /// Events processed in this workflow generation. Reset on continue-as-new.
    pub events_so_far: u64,
}

#[workflow]
pub struct CdcPipelineWorkflow {
    input: CdcPipelineInput,
}

fn opts_short() -> ActivityOptions {
    ActivityOptions {
        start_to_close_timeout: Some(Duration::from_secs(60)),
        ..Default::default()
    }
}
fn opts_long() -> ActivityOptions {
    ActivityOptions {
        start_to_close_timeout: Some(Duration::from_secs(600)),
        ..Default::default()
    }
}

#[workflow_methods]
impl CdcPipelineWorkflow {
    #[init]
    pub fn new(_ctx: &WorkflowContextView, input: CdcPipelineInput) -> Self {
        Self { input }
    }

    #[run]
    pub async fn run(ctx: &mut WorkflowContext<Self>) -> WorkflowResult<()> {
        let input = ctx.state(|s| s.input.clone());
        let pg = match &input.spec.source {
            SourceSpec::Postgres(p) => p.clone(),
            _ => return Err(WorkflowTermination::failed(anyhow::anyhow!(
                "CdcPipelineWorkflow requires Postgres source"))),
        };
        let rel_name = format!("{}.{}", pg.schema, pg.table);

        // 1. Ensure slot (idempotent). First-launch only — subsequent
        //    continue-as-new re-uses the cached slot metadata.
        let slot = match input.slot_ready.clone() {
            Some(s) => s,
            None => ctx
                .start_activity(
                    CdcActivities::ensure_slot,
                    EnsureSlotInput {
                        pipeline_id: input.pipeline_id,
                        source_url: input.source_url.clone(),
                        schema: pg.schema.clone(),
                        table: pg.table.clone(),
                    },
                    opts_short(),
                )
                .await?,
        };

        // 2. Snapshot — only on first launch (no resume_lsn).
        if input.resume_lsn.is_none() {
            ctx.start_child_workflow::<CdcSnapshotWorkflow, _, _>(
                CdcSnapshotInput {
                    pipeline_id: input.pipeline_id,
                    run_id: input.run_id,
                    source_url: input.source_url.clone(),
                    schema: pg.schema.clone(),
                    table: pg.table.clone(),
                    pk_col: pg.pk_columns.first().cloned()
                        .ok_or_else(|| WorkflowTermination::failed(
                            anyhow::anyhow!("CDC requires at least one PK column")))?,
                    batch_size: input.spec.batch_size.max(100),
                    consistent_point: slot.consistent_point.clone(),
                    destination: input.spec.destination.clone(),
                },
                ChildWorkflowOptions::default(),
            )
            .await?
            .result()
            .await?;
        }

        // 3. Streaming loop.
        let mut current_lsn: Option<String> = input
            .resume_lsn
            .clone()
            .or_else(|| Some(slot.consistent_point.clone()));
        let mut events_so_far = input.events_so_far;
        let mut batch_seq: u32 = 0;
        loop {
            let out: ReadWindowOutput = ctx
                .start_activity(
                    CdcActivities::read_window,
                    ReadWindowInput {
                        pipeline_id: input.pipeline_id,
                        run_id: input.run_id,
                        batch_seq,
                        source_url: input.source_url.clone(),
                        slot_name: slot.slot_name.clone(),
                        publication_name: slot.publication_name.clone(),
                        start_lsn: current_lsn.clone(),
                        max_events: 1000,
                        max_wait_ms: 2_000,
                        table_rel_name: rel_name.clone(),
                        destination: input.spec.destination.clone(),
                    },
                    opts_long(),
                )
                .await?;
            events_so_far = events_so_far.saturating_add(out.rows as u64);
            if let Some(lsn) = out.new_lsn {
                current_lsn = Some(lsn);
            }
            batch_seq += 1;

            if events_so_far >= CAN_EVENT_BUDGET {
                let mut next = input.clone();
                next.resume_lsn = current_lsn.clone();
                next.slot_ready = Some(slot.clone());
                next.events_so_far = 0;
                return Err(WorkflowTermination::continue_as_new(
                    ContinueAsNewWorkflowExecution::new::<CdcPipelineWorkflow, _>(next)
                        .map_err(WorkflowTermination::failed)?,
                ));
            }
            // Empty window → sleep to avoid tight spin when idle.
            if out.rows == 0 {
                ctx.sleep(Duration::from_secs(2)).await;
            }
        }
    }
}
```

## Appendix A — Operational notes you will need while executing

**Pre-existing slot/publication blocking CREATE.** If a prior run crashed before releasing, you'll see `ERROR: replication slot "etl_…" already exists` or `publication "etl_…_pub" already exists`. Both drops are idempotent:

```bash
docker exec etl-postgres psql -U etl -d cdc_source_demo -c \
  "SELECT pg_drop_replication_slot(slot_name) FROM pg_replication_slots WHERE slot_name LIKE 'etl_%';"
docker exec etl-postgres psql -U etl -d cdc_source_demo -c \
  "DO \$\$ DECLARE r record; BEGIN FOR r IN SELECT pubname FROM pg_publication WHERE pubname LIKE 'etl_%' LOOP EXECUTE format('DROP PUBLICATION %I', r.pubname); END LOOP; END \$\$;"
```

**REPLICA IDENTITY matters for updates/deletes.** Without `FULL` (or at least `USING INDEX <pk>`), pgoutput only emits the PK in the before-image for updates and the key for deletes. The MVP seed sets `REPLICA IDENTITY FULL` for simplicity; production pipelines would document the tradeoff.

**tokio-postgres keepalive replies.** The server sends periodic `PrimaryKeepaliveMessage` (`k`) asking the consumer to confirm progress. MVP ignores them (slot won't be garbage-collected because we hold it via `START_REPLICATION`). Production needs proper standby-status replies — deferred but noted.

**Transaction demarcation.** MVP does not preserve transaction boundaries. Every Insert/Update/Delete inside a source transaction becomes a separate Arrow row; they share `_cdc.txid` so consumers can reconstruct if needed.

**pgoutput protocol version.** This plan targets proto_version 1, which Postgres 10+ supports. Proto 2 (streaming of in-progress txns) is Postgres 14+ and out of scope.

**Continue-as-new tuning.** `CAN_EVENT_BUDGET = 10_000_000` is intentionally conservative for MVP. The streaming loop does not currently continue-as-new on wall-clock or activity-count budgets — adding those is straightforward once we see how long a window takes in practice.
